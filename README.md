# Milan Explorer

A travel guide website for Milan, Italy - built as a front-end web development project. Whether you're planning a trip or just curious about the city, Milan Explorer walks you through the must-see spots, where to eat, how to get around, and where to escape to for a day.

---

## What's inside

| Page | What you'll find |
|---|---|
| **Home** | Intro slideshow + contact form |
| **Attractions** | Duomo, Sforzesco Castle, Navigli, Brera, The Last Supper |
| **Food & Drink** | Breakfast culture, aperitivo, pasta, pizza, gelato |
| **Getting Around** | Metro lines, trams, contactless payment, cycling |
| **Day Trips** | Genoa, Lecco (Lake Como), and Turin |
| **Safety Tips** | Practical advice for staying safe in the city |

---

## Built with

- **HTML5** - semantic, commented markup across 6 pages
- **CSS3** - custom stylesheet with responsive layout and hover effects
- **JavaScript** - handles the image slideshow, scroll-reveal animations, info popups, and contact form validation

---

## How to run it

No build tools, no setup - just open it in a browser.

```bash
git clone https://github.com/zudiie/milan-explorer.git
cd milan-explorer
open index.html
```

Or just double-click `index.html`

## Project structure

```
milan-explorer/
├── index.html              # Home - slideshow, intro, contact form
├── attractions.html        # Must-see sights around Milan
├── day_trips.html          # Day trips to Genoa, Lecco & Turin
├── food_drink.html         # Food and drink guide
├── getting_around.html     # Transport options and ticketing

├── script.js               # Slideshow, animations, form logic
└── images/                 # Photos used across the site
```

---

## Features

- Auto-advancing image slideshow with dot and arrow navigation
- Scroll-triggered reveal animations on the Attractions and Food pages
- Clickable info popups on the Food & Drink page
- Contact form with client-side validation and success/error feedback
- Fully responsive - works on mobile, tablet, and desktop
